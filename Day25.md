🦈 AI Shark Tank Simulator (Day 25)
A complete, production-quality, client-side web application that simulates a high-stakes startup pitch room. Users present their company dossier to a panel of 4 distinct, dynamic AI Judge personas, answer tough strategic questions in real time, and receive an automated financial evaluation report.

🚀 Key Features
Dynamic AI Roster: Face 4 interactive personas (Venture Capitalist, Serial Founder, Target Customer, Angel Investor) each tracking separate valuation parameters (TAM, execution moats, customer friction, and unit economics).

Interactive 8-Round Q&A: A real-time chat interface where judges react contextually to the length, metrics, and depth of your strategic answers.

Algorithmic Evaluation Matrix: Calculates live composite scores across Market Potential, Innovation, Business Model, and Execution.

Investment Decision Engine: Generates structured verdicts (Invest, Acquire, Defer, or Reject) with calculated valuations, funding metrics, and written deliberation briefs.

Production-Ready UI/UX: Built using Tailwind CSS with an immersive dark-theme dashboard layout, smooth animations, interactive modals, and persistent local storage leaderboards.

Client-Side Export Tooling: Integrated with Canvas Confetti for successful raises and html2pdf.js for instant local PDF Pitch Report downloads with no backend requirement.
